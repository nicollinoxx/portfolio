Sobre o projeto

O Depot é uma aplicação de loja virtual criada como parte de um tutorial de aprendizado em Rails. Esse projeto foi desenvolvido para explorar e implementar funcionalidades essenciais de um e-commerce, como gerenciamento de produtos, carrinho de compras, e processamento de pedidos. O código-fonte está disponível no repositório Learning-storage no meu perfil do GitHub.
Caso tenha mais interesse sobre o projeto visite o post E-commerce "Depot".
