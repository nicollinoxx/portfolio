Sobre o projeto

GeoMap é um aplicativo web que permite aos usuários criar e visualizar localizações em um mapa interativo. Os usuários podem adicionar lugares, que são marcados com pins personalizados no mapa. Além disso, o sistema pode armazenar imagens e informações sobre cada local.

O projeto utiliza MapLibre, uma biblioteca open-source para renderização de mapas interativos, permitindo uma experiência fluida e personalizável sem dependências de serviços pagos.
